# Tournament Management System

A college-level web-based Tournament Management System developed using **JSP, Servlets, JDBC, MySQL, and Apache Tomcat 9**.  
The system allows administrators to manage tournaments, teams, matches, results, and points tables, while normal users can register teams and view tournament details.

---

## Project Overview

The Tournament Management System is designed to simplify the management of sports or college-level tournaments.  
It provides separate dashboards for **Admin** and **User** roles.

The Admin can create tournaments, schedule matches, update results, and view teams.  
The User can register teams, view tournaments, check match schedules, and view points tables.
